## AJAX
***Progetto Photo Rover AJAX***

Il codice è stato scritto per ricavare delle informazioni dall'API del "Photo Rover Mars". Tramite questo codice vengono estratti i nomi delle fotocamere dalle quali sono state scattate le foto, i relativi id unici e la data dello scatto.
Il bottone, dopo esser stato cliccato, scompare perchè non ha più informazioni da mostrare e, allo stesso tempo, vengono rilasciate in colonna le informazioni.
